A multilingual application built on Next.js, Tailwind CSS, and TypeScript.

## Getting Started

1. **Clone and Install**
```bash
git clone <repository-url>
cd bookstore
npm install
```

2. **Run Development Server**
```bash
npm run dev
```

3. **Open [http://localhost:3000](http://localhost:3000)**
